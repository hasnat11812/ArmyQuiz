# Optional: HTML -> PDF generation using WeasyPrint.
# NOTE: WeasyPrint depends on native libraries (cairo, pango, gdk-pixbuf, libffi, etc.)
# which must be installed on the host OS before pip installing the Python package.
#
# Debian / Ubuntu example (Linux):
# sudo apt update; sudo apt install -y build-essential libcairo2 libpango-1.0-0 \
#   libgdk-pixbuf2.0-0 libffi-dev shared-mime-info libjpeg-dev libpng-dev
#
# Fedora / CentOS example (Linux):
# sudo dnf install -y cairo pango gdk-pixbuf2 libffi shared-mime-info libjpeg-turbo-devel libpng-devel
#
# Windows notes:
# - On Windows, the easiest approach is to install the GTK runtime (MSYS2 or GTK bundles)
#   or use an alternative rendering approach (wkhtmltopdf or headless Chromium via Playwright).
# - If you want to try WeasyPrint on Windows, install GTK/cairo dependencies via MSYS2
#   or rely on a prebuilt distribution. See WeasyPrint docs for Windows specifics.
#
# If you have the native deps installed, uncomment the package below and install via pip.
